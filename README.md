# نظام المهام التعاوني 🤝

تطبيق ويب يتيح للمستخدمين إنشاء المهام ومشاركتها مع مستخدمين آخرين.

## كيفية التشغيل محلياً
1. ثبت الاعتماديات: `cd backend && npm install`
2. أنشئ قاعدة بيانات MySQL وشغّل ملف `backend/database/schema.sql`
3. انسخ `.env.example` إلى `.env` وعدل الإعدادات
4. شغل الخادم: `cd backend && npm start`
5. افتح المتصفح على: `http://localhost:3000`
